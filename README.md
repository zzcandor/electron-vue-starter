# electron-vue-start
基于electron和vue2.0搭建的electron starter项目，方便项目快速初始化，开箱即用。
## Project setup
```
npm install
```

### 本地预览
```
npm run  electron:serve
```

### 打包应用
```
npm run  electron:build
```

### Lints and fixes files
```
npm run lint
```

## vue项目添加electron步骤

```
vue create  my-project
vue add electron-builder
```
